About ECG Arrhythmia Classifier

Overview

This application uses advanced deep learning techniques to classify different types of cardiac arrhythmias from ECG spectral images. The system is trained on a comprehensive dataset and can accurately identify 6 different types of heart conditions.

Supported Arrhythmia Types

  • Left Bundle Branch Block (LBBB) - Electrical conduction delay in the left ventricle
  • Normal - Normal heart rhythm
  • Premature Atrial Contraction (PAC) - Extra heartbeats originating in the atria
  • Premature Ventricular Contractions (PVC) - Extra heartbeats originating in the ventricles
  • Right Bundle Branch Block (RBBB) - Electrical conduction delay in the right ventricle
  • Ventricular Fibrillation - Chaotic electrical activity in the ventricles

Technical Details

  • Deep Convolutional Neural Network (CNN) architecture
  • Trained on 2D ECG spectral image representations
  • Advanced image preprocessing and augmentation
  • Real-time classification with confidence scores
